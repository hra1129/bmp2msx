2) Input file
    - Windows BMP File 1bit
    - Windows BMP File 4bits
    - Windows BMP File 8bits
    - Windows BMP File 16bits
    - Windows BMP File 24bits
    - Windows BMP File 32bits
    - Clipboard BMP picture.
    And Susie PlugIn supported file.

3) Output file ( Picture )
    - SCREEN2(256x192x16colors)
    - SCREEN3(64x48x16colors)
    - SCREEN5(256x212(192)x16colors)
    - SCREEN5Interace(256x424(384)x16colors)
    - SCREEN7(512x212(192)x16colors)
    - SCREEN7Interace(512x424(384)x16colors)
    - SCREEN8(256x212(192)x256colors)
    - SCREEN8Interace(256x424(384)x256colors)
    - SCREEN10(256x212(192)x12499colors )
    - SCREEN10Interace(256x424(384)x12499colors )
    - SCREEN12(256x212(192)x19268colors )
    - SCREEN12Interace(256x424(384)x19268colors )

4) Output file ( Palette )
    - type of MSX-BASIC COLOR=RESTORE
    - type of (C)BIT2 Graphsaurus v2.x PL5/PL7
    - type of CSV text ( , )

5) Run 'BMP to MSX'
  Then double click activates 'BMP2MSX.EXE'.
